🧪 OrangeHRM Playwright Automation Testing

📌 Project Description

This project focuses on automating the OrangeHRM web application using Playwright with JavaScript. The automation framework 
validates core HR modules such as login, employee management, PIM, leave management, and user role workflows. The project 
follows the Page Object Model (POM) design pattern to maintain clean and reusable test code and includes detailed test reporting.
